imshow显示图像为纯白

图像数据的标准类型是uint8，当图像参与运算时，为了保证精度，会将其从uint8类型转换为double类型（类型变化，数值不变），double型的范围在0~1，超过1的当做1，所以用imshow显示图像时是一片白色。

解决方法：

  1. 用uint8()对其进行类型转换；
  2. 影像/255，将其转换到0~1；
  3. imshow(I,[])，自动调整数据的范围以便显示；
