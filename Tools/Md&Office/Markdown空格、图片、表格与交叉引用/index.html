实现段首空格的显示

方法一

「Markdown 语言」不负责实现段首缩进 (ˊ_>ˋ)

段首缩进这件事，应该是 CSS 或者其他排版工具的事情，Markdown 奉行的是样式和内容分开的哲学。

即使手动输入空格，  也是很不推荐的方法。我推荐全角空格，切换到全角模式下（一般的中文输入法都是按 shift + space）输入两个空格就行了。这个相对   来说稍微干净一点，而且宽度是整整两个汉字，很整齐。

表格

简单表格

直接使用Markdown自带的表格可以快速建立，但是对表格缺少设置，如果需要设置比较复杂的表格建议使用table。

效果：

表头1表头2
12

TML table 表格

<tabel>：表格声明
<tbody>：整个表格加载结束以后显示
<th>：表头
<td>：表格内容
colspan="2"：占用两列
rowspan="2"：占用两行
bgcolor="Pink"：背景颜色

显示结果：

表头1表头2colspan="2"
rowspan="2"21bgcolor="Pink"41
223242
